# 思帆商 SIFANGSHANG 企业官网 — 设计方案

**日期:** 2026-06-21
**状态:** 已实现（3个版本设计页面已完成开发）

---

## 1. 项目概述

思帆商（SIFANGSHANG）是一家专注于静电防尘领域的企业品牌，产品涵盖防静电服装、鞋子、口罩、椅子、地板、包装及工程服务，主要服务电子厂、实验室、生物科技企业。

本设计提供3个差异化版本的企业官网首页，每个版本都是完整可交互的页面。

基于对ESD行业企业官网的调研，参考了以下竞品：

### 设计趋势总结

- **配色**：科技蓝主导，安全绿/橙色作CTA强调，大量留白
- **布局**：F型视觉流，Hero区+优势数字+产品卡片+解决方案+CTA
- **内容策略**：从产品目录转向解决方案导向，场景化呈现
- **交互**：视差滚动、微交互动效、数据可视化

## 3. 设计方案

### Version A: Tech-Forward Premium

**定位**：高端ESD整体解决方案提供商
**配色**：深海蓝(#0A1628) + 电光蓝(#0066FF) + 白色 + 琥珀色CTA
**设计特点**：
- 全屏Hero区带动画背景（网格+渐变球体+扫描线）
- Hero区右侧浮动的ESD/静电控制卡片，营造科技感
- 数据统计条（经验/客户/面积/满意度）
- 产品中心6卡片网格
- 3列行业解决方案卡片（电子制造/实验室/生物科技）
- 工程服务左右分栏，带勾选列表
- 认证资质展示条
- 深色CTA表单区

### Version B: Clean Industrial

**定位**：洁净、高端的精密防护品牌
**配色**：白色 + 青绿(#00A896) + 浅灰(#F5F7FA)
**设计特点**：
- 极简Hero区，几何圆环装饰
- 4列icon-row功能特性
- 大画幅产品卡片（彩色背景+大ICON+描述）
- 工程服务步骤编号（01-04）
- 3列客户案例卡片
- 简约联系方式表单
- 大量留白，瑞士国际主义风格

### Version C: Bold Factory

**定位**：制造实力派，工程技术导向
**配色**：深灰(#1E1E2E) + 橙色(#FF6B35) + 白色
**设计特点**：
- 粗体大字排版（font-black）
- 全出血位视觉区块，对角线切割
- 生产能力数据展示（面积/产线/产能/人员）
- 非对称产品网格布局
- Industry Tab切换内容区
- 编号式工程服务卡片
- 角标/标签设计语言
- 工业粗野主义融合现代网页设计

## 4. 技术实现

- **框架**: Next.js 16 + TypeScript
- **样式**: Tailwind CSS 4
- **组件**: 共享Header（含响应式移动菜单）、Footer、ProductCard、SolutionCard
- **路由**:
  - `/` — 版本选择首页
  - `/v1` — Tech-Forward Premium
  - `/v2` — Clean Industrial
  - `/v3` — Bold Factory

### 共享组件

| 组件 | 路径 | 说明 |
|------|------|------|
| Header | `src/components/Header.tsx` | 导航栏，支持default/dark/transparent三种样式，中英双语切换，移动端汉堡菜单 |
| Footer | `src/components/Footer.tsx` | 页脚，产品/方案/关于三栏链接 + 品牌信息 |
| ProductCard | `src/components/ProductCard.tsx` | 产品卡片，支持4种配色变体 |
| SolutionCard | `src/components/SolutionCard.tsx` | 解决方案卡片，支持3种渐变色 |

### 全局样式

`src/app/globals.css` 包含：
- Tailwind v4主题配置（品牌色系统）
- 动画定义（fadeInUp, slideInLeft, float, pulse-glow, gradient-shift等）
- 通用工具类（glass, glass-dark, gradient-text, reveal等）
- 自定义滚动条样式
